# LanTalk - Chat with peers on local network

With LanTalk, you can chat with anyone who is connected on the same network via WIFI.

![LanTalk](lantalk.png)

## How to run

- npm install
- npm run dev

## How to package

#### For Windows

- npm run build:win

#### For Mac

- npm run build:mac

#### For Linux

- npm run build:linux

NOTE: This assumes `electron-packager` is installed globally
